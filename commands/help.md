Invoke the `help` skill to display plugin reference information.

**Modes:**

- `/help` — condensed command list (one-liner per command) plus a tip to use `/help <command>` for detail
- `/help commands` — full command table with purpose
- `/help <command>` — detail for a specific command: description, what it does, and a usage example
